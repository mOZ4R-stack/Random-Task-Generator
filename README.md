# Random Task Generator

**Автор:** Осокин Павел

## Описание

Приложение для генерации случайных задач с графическим интерфейсом. Помогает разнообразить день, случайно выбирая задачу из списка. Поддерживает фильтрацию по типу задач (учёба, спорт, работа), сохраняет историю сгенерированных задач и позволяет управлять списком задач.

## Возможности

- 🎲 Генерация случайной задачи
- 📋 Предопределённый список задач
- 🔍 Фильтрация по типу (учёба/спорт/работа)
- 📜 История всех сгенерированных задач с датой
- ➕ Добавление новых задач с проверкой на пустую строку
- ✖ Удаление задач
- 💾 Автоматическое сохранение в JSON (задачи и история)
- 🎨 Интуитивно понятный интерфейс
